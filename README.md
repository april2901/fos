Focus on Speaking App UI
========================

로컬 실행 방법
-------------

1. 의존성 설치  

    npm install

2. 개발 서버 실행  

    npm run dev  

    터미널에 표시되는 주소 (예: http://localhost:5173)를 브라우저에서 열면 됩니다.

3. 프로덕션 빌드 및 미리보기  

    npm run build  
    npm run preview
